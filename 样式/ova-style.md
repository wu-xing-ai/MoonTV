# OVA.ai 网站风格与动画效果总结

## 一、整体风格定位

OVA.ai 是一家加拿大 AI 沉浸式技术公司，网站整体呈现 **暗色科技风 + 霓虹蓝紫色调**，风格偏向未来感、高端 B2B 营销站点。基于 **Webflow** 构建。

---

## 二、色彩体系

| 用途     | 色值                        | 说明                                   |
| -------- | --------------------------- | -------------------------------------- |
| 主色调   | `#5263FF`                   | 鲜明的蓝紫色，用于按钮、链接、强调元素 |
| 高亮色   | `#4DC6FC`                   | 青色，用于选中态、悬停效果             |
| 深色背景 | `#313136`                   | 炭灰色，滚动条轨道、深色区块           |
| 文字     | 白色 on 深色 / 深色 on 浅色 | 高对比度阅读体验                       |

整体配色方案：**Dark Mode 为主**，深灰/近黑背景搭配蓝紫霓虹点缀，营造高科技沉浸感。

---

## 三、字体排版

| 字体           | 用途          | 字重范围       |
| -------------- | ------------- | -------------- |
| **Montserrat** | 标题/展示字体 | 100–900 + 斜体 |
| **Roboto**     | 正文/UI 文字  | 300–700        |
| **Sacramento** | 装饰性手写体  | 300–700        |

字体加载方式：`WebFont.load()` 异步加载 Google Fonts。

排版特点：标题大字号、粗字重，正文细字重，形成强烈视觉层次。

## 四、核心动画效果

### 1. Logo 无限滚动轮播（Marquee）

- 合作伙伴 Logo 水平无限滚动
- 通过 **DOM 复制** 实现无缝循环
- 速度适中，不打断阅读

### 2. Hero 区域浮动关键词药丸（Pills）

- 五个关键词（VR, AR, no-code, autonomy, real-time intelligence）
- 每个 Pill 带 "+" 图标前缀
- 悬浮感 + 缓动动画

### 3. 滚动触发动画（Scroll-triggered）

- 使用 Webflow Interactions 2.0
- 元素在滚动进入视口时渐入（fade-in）/ 滑入
- 适度使用，不喧宾夺主

### 4. 案例轮播（Case Study Carousel）

- 左右箭头按钮导航
- 卡片式布局，含缩略图 + 标题 + 描述
- 平滑滑动过渡

### 5. 悬停交互

- 卡片 ">>" 指示器悬停变色
- 按钮 hover 态颜色变化
- 链接下划线动画

### 6. 语言切换动画

- `toggleDivsBasedOnURL()` 切换英语/法语内容
- 基于 URL 路径 (`/fr/`) 触发

## 八、风格关键词总结

> **Dark Mode** · **霓虹蓝紫** · **沉浸感** · **科技未来感** · **B2B 高端** · **滚动驱动动画** · **无限轮播** · **卡片式布局**

OVA.ai 的设计语言核心是：**深色背景 + 蓝紫霓虹点缀 + 克制的滚动动画**，在高端科技感与信息可读性之间取得平衡，不追求花哨特效，而是以精确的视觉层次和微妙的动效传递品牌的专业与前沿定位。

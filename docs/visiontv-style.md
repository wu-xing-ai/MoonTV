# Vision TV 网站风格与设计总结

## 一、整体风格定位

Vision TV 是一个影视聚合流媒体平台，网站风格呈现 **现代简约 + 玻璃拟态（Glassmorphism）**，支持亮色/暗色双主题。整体偏向内容驱动的消费级产品，注重内容展示密度与浏览效率。基于 **Next.js + Tailwind CSS** 构建。

---

| 用途         | 色值                       | 说明              |
| ------------ | -------------------------- | ----------------- |
| 页面背景     | `#000000`                  | 纯黑              |
| 侧边栏背景   | `rgba(17,24,39,0.7)`       | 半透明深灰 + 模糊 |
| 侧边栏边框   | `rgba(55,65,81,0.5)`       | 深灰分割线        |
| 主文字       | `#E5E7EB` (gray-200)       | 浅色正文          |

## 三、字体排版

| 字体      | 用途     | 说明                                  |
| --------- | -------- | ------------------------------------- |
| **Inter** | 全局字体 | Tailwind 默认，系统级 sans-serif 回退 |

- 标题字号：`20px`，字重 `700`
- 正文/导航：`16px`，字重 `500`
- 辅助文字：`14px`（"查看更多"等）
- 评分/标签：`12px`

排版特点：单一字体族，完全通过字重和字号制造层次，无装饰性字体。

## 五、核心 UI 组件

### 1. 侧边栏（Sidebar）

- **玻璃拟态**：`backdrop-filter: blur(20px)` + 半透明背景
- 宽度：展开 `256px` / 折叠 `64px`
- 顶部 Logo：图标 + 站名，绿色品牌色
- 导航项：图标 + 文字，活跃项绿色背景高亮
- 可折叠：汉堡按钮切换，状态保存到 `localStorage`

### 2. 海报卡片（Movie Card）

- 尺寸：`180px × 270px`（2:3 比例，标准海报）
- 容器：`border-radius: 8px`，`overflow: hidden`
- 评分角标：绝对定位左上角，绿色背景
- 集数进度：绝对定位右上角（如 "21/23"）
- 标题：卡片下方显示

### 3. 水平滚动区域

- 每个内容板块使用水平滚动（`overflow-x: auto`）
- 两侧渐变遮罩，暗示可滚动
- 右侧 "查看更多" 链接

### 4. 继续观看卡片

- 水平滚动，16:9 宽屏缩略图
- 叠加集数进度文字
- 显示来源（如 "卧龙资源"）

### 5. Tab 切换（首页/收藏夹）

- 活跃 Tab：白色/亮色文字（`font-weight: 500`）
- 非活跃 Tab：灰色文字（`gray-400`）
- 无下划线，纯文字对比

### 6. 主题切换

- 明/暗一键切换
- 通过 `<html class="dark">` 控制
- 所有颜色跟随 Tailwind `dark:` 前缀自动切换

## 六、动画与过渡

| 效果              | 实现                                        |
| ----------------- | ------------------------------------------- |
| 侧边栏展开/折叠   | `transition-all duration-300` 宽度动画      |
| 导航项悬停        | 背景色渐变 `transition-colors duration-200` |
| 侧边栏展开时 Logo | `opacity` 渐入渐出 `duration-200`           |
| 主题切换          | 颜色瞬切（依赖 Tailwind dark 类切换）       |
| 卡片/链接悬停     | `hover:opacity-80 transition-opacity`       |
| 侧边栏背景        | `backdrop-blur-xl` 持续模糊                 |

全站共约 573 个元素带 transition，过渡属性主要涉及：

- `color, background-color, border-color` — 颜色过渡
- `opacity` — 透明度过渡
- `all` — 通用过渡（侧边栏等）
- `transform` — 位移动画

**动画风格**：克制实用，无花哨特效，200-300ms 快速过渡。

| 维度     | OVA.ai                       | Vision TV                |

## 九、风格关键词总结

> **Glassmorphism** · **双主题** · **绿色品牌** · **水平滚动** · **海报墙** · **侧边栏导航** · **Tailwind 原子化** · **Inter 字体** · **内容密度优先** · **克制过渡**

Vision TV 的设计语言核心是：**玻璃拟态侧边栏 + 高密度水平滚动海报墙 + 单一绿色强调色**，在视觉品质与浏览效率之间取得平衡，不追求炫技，而是以清晰的信息层级和流畅的交互体验服务于内容消费场景。
